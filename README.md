## Dataset
The dataset contains 3064 pairs of MRI brain images and their respective binary mask indicating tumor.
<br/> <br/>
Dataset: [Brain Tumor Segmentation](https://figshare.com/articles/dataset/brain_tumor_dataset/1512427)
